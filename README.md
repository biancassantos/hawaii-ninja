<h1 align="center">Hawaii Ninja</h1>

Hawaii Ninja é um jogo 2D single-player desenvolvido na linguagem Python utilizando a biblioteca do PyGame. Ele foi desenvolvido como atividade prática da disciplina Linguagem de Programação Aplicada.

## História
Inimigos ameaçam a vida tranquila no Hawaii. 
Polvos do mal e Águas-vivas sinistras espalham o medo pela região e, à noite, recebem ajuda das vilãs mais perigosas de todas: as Górgonas! 
Você consegue ajudar a ninja Kunoichi a manter o arquipélago seguro?
